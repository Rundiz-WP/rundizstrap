== Description ==
Based on the latest version of Bootstrap, it is a WordPress Full Site Editor (FSE) theme.

The concept is to keep it basic, minimal, and support most Bootstrap features.

This theme recommend plugin "rundizstrap-companion". You can download it from https://wordpress.org/plugins/rundizstrap-companion/ or https://github.com/Rundiz-WP/rundizstrap-companion

Tested up to PHP 8.5.

= Expand/collapse contents help (read more/read less) =
To make content's height limited, add the class `rundizstrap-entry-content-readmore` to the post content block that contains the CSS `entry-content`, and this functionality will be activated automatically.
